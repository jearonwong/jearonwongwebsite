# JearonWong.com VI & UI/UX System Brief v0.1

This document outlines the visual identity and interface primitives that stem from the Jearon Wong Brand Identity Brief. The goal is to create an aesthetically memorable, structured, and premium AI-era personal brand surface.

## Layer 2 — Visual Identity (视觉识别 / VI)

The visual layer translates the "Founder/Architect" persona into color, typography, imagery, and layout patterns.

- **Visual Keywords:** editorial-tech, modern, high-contrast, premium, intelligent, sharp, AI-era but not cliché.
- **Color System:**
  - *Base Dark:* Near-black (e.g., deep charcoal or slate black).
  - *Base Light:* Off-white / cool light (reduces eye strain, looks sophisticated rather than raw #FFF).
  - *Accent:* One restrained cool accent (e.g., electric blue, icy cyan, or strict monochromatic interplay—pending palette exploration).
- **Type System:**
  - *Primary Display:* Modern sans / grotesk (e.g., Inter, Roboto Mono for numbers, or Outfit/Geist for modern tech feel).
  - *Editorial Accent:* Selective serif *if* needed (for blockquotes or essay leads).
  - *Micro-labels:* Monospace (for UI tags, project status, dates).
- **Image System:**
  - *Portrait Style:* High-contrast, editorial crop, premium, serious, founder / architect energy. 
- **Graphic Motifs:**
  - Visible grid systems (aligning with "protocol/architecture").
  - Node/signal traces.
  - Protocol / system fragments.
  - Subtle diagrammatic texture (no cliché robotic/AI shiny blobs).
- **Layout Language:**
  - Split hero structures.
  - Large, intentional whitespace.
  - Asymmetry with strict underlying control.
  - Strong section rhythms (breaking monotony).
  - Fewer, but deliberately curated, equal-weight cards.

## Layer 3 — UI System

The building blocks drawn from the VI to construct the interface.

- **Primitives:** Button, link button, card, signal card, project proof card, featured article card, portrait frame, eyebrow, lead text, badge, metadata row, section shell, split layout, narrative band.
- **Constraints:**
  - All repeated UI elements must be strictly componentized.
  - No one-off visual drift allowed; sticking to standard token values.
  - Tokens must drive spacing, radius, color, and typography entirely.

## Layer 4 — UX / Experience Flow

How a user navigates and consumes the brand story, starting from the decisive first 10 seconds.

- **Homepage Narrative Flow:**
  1. The Person
  2. The Thesis
  3. Signature Ideas
  4. Why Now
  5. The Proof
  6. Featured Writing
  7. Playbooks
  8. Subscribe / Contact Path
- **Recognition Goals (Time-to-Value):**
  - *3 Seconds:* Remember the person (High-impact portrait + Brand Name).
  - *5 Seconds:* Understand the thesis (Clear, unmissable thesis line).
  - *10 Seconds:* See the proof chain (Immediate transition downward into MPLP / Cognitive OS evidence).
- **Behavioral Goals:**
  - Continue reading / scrolling.
  - Explore projects (not just reading, but inspecting artifacts).
  - Read flagship essay.
  - Subscribe to the stream.
